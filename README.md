FOC Basic Algorithm Examples

This project is a collection of FOC driver example programs based on the STM32G431CBT6, providing a complete set of entry-level FOC examples. It includes basic FOC algorithms such as open-loop speed control, angle measurement, closed-loop position control, current measurement, closed-loop current control, speed measurement and filtering, closed-loop speed control, and closed-loop position-speed-torque control.

Development Environment

Peripheral configuration is performed using STM32CubeMX, while Keil is used for programming and flashing. The MCU model is STM32G431CBT6.
